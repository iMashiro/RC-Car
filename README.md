# WiFi RC-Car
<p>Project of a WiFi RC-Car with esp8266 + L293D Motor Shield</p>

<p>The project resumes in a Wifi RC-Car controlled with blynk app.</p>
<p>I used 2x9V battery with 450mAh.</p>
